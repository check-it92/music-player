** API
API(애플리케이션 프로그래밍 인터페이스)는 소프트웨어 프로그램들이
서로 통신할 수 있도록 하는 규칙과 프로토콜을 의미합니다. 

API의 특징
소프트웨어 프로그램 간의 연결을 가능하게 합니다 
애플리케이션 개발을 간소화하여 시간과 비용을 절약할 수 있습니다 
기존 애플리케이션 기능을 새 애플리케이션에 통합할 수 있습니다 
인터넷을 통해 다양한 정보를 검색하고 이용하는 데 활용됩니다 
모바일 앱이나 IoT 앱이 서버와 데이터 통신을 위해 사용됩니다 

api 는 데이터를 무한으로 주지 않기 때문에 그 부분 조심해서 사용해야함
